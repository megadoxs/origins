Origins

The default Origins:

Origins is a minecraft mods that gives every player the opportunity to choose his origin (similar to a race) and have powers that reflect that origin.
If this was all the Origins mod had it wouldn't have been so great since as you will see, there is only 6 origins by default.
You may or may not understand now that I told you that there is 9 orgins by default, that you can have more.
As I will show you later, this mods not only adds hard codded orgins, but also give the opportunity to make your own.
Using the stadart minecraft datapacks format and the special tools that Origins give you, making custom origins could not be easier.

Personalised Origins

Now here is the part that is really great about this minecraft mod!
*To keep things easily understandable, I won't be presenting all the tools that are avaiable, simply know that there is more.
So, Here you are, wanting to create a custom origin to play with your friends.
Well you are going to need to give this origin powers.
Here is the big category that you need to know and what they do.

  • Power Types

    power types are in a way a countainer that you can only be access if the general condition for the power type happend.
    Might be confusing so here is an example:
    You want to do something every 10 seconds, the best way to do so would be to use a action over time that does wathever is inside this "power type" every specified amount of time.
    This power type would then execute Actions that you have put in it if the Conditions that you gave the "power type" to execute are true.

  • Actions

    actions are events that you want to happend when the power types executes, which they are part of.
    e.g. you want to brun the player every 10seconds.
    you would use an action over time power type make it so that it executes every 10seconds and use a burn entity action.

  • Conditions

    Conditions are well conditions that you want to be true before the power types executes, which they are also part of.
    e.g. you want to brun the player every 10seconds if the player is exposed to sun.
    you would use an action over time power type make it so that it executes every 10seconds and use a burn entity action and an exposed to sun entity condition.
